# APD

Temele de la Algoritmi Parareli si Distribuiti 2021-2022. Au ca scop dezvoltarea unor aplicatii pararele (temele 1, 2) si distribuite (tema 3), implementate in **C** si **Java**.

## Teme

* Tema 1 - Pararelizarea unui algoritm genetic in **C** folosind **Pthreads**.

* Tema 2 - Implementarea unui pipeline **Map-Reduce** pentru rezolvarea unei probleme de procesare de fisiere text in **Java**.

* Tema 3 - Implementarea unui program distribuit de rezolvare a unei probleme de calcul folosind toate nodurile din cadrul unei topologii prestabilite folosind **MPI** in **C**.

Enuntul complet si detaliile complete de implementare se gasesc in folderele asociate temelor.
